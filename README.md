# Data Analytics Project

## Overview

This project demonstrates an end-to-end **data analytics workflow** using Python, SQL, and Power BI.
The goal of the project is to load a dataset, perform **exploratory data analysis (EDA)**, clean and prepare the data, analyze it using **SQL queries on PostgreSQL**, and create an **interactive Power BI dashboard** to present insights.

The final results are summarized in a **report and presentation (created using Gamma)** to communicate findings clearly.

---

## Dataset

The dataset used in this project contains information related to **customer shopping behavior**, including:

* Customer ID
* Age
* Gender
* Item Purchased
* Category
* Purchase Amount
* Location

This dataset helps analyze **customer purchasing patterns, product preferences, and spending behavior**.

---

## Tools & Technologies

The following tools and technologies were used:

* **Python** – Data loading, cleaning, and analysis
* **Pandas** – Data manipulation and EDA
* **Jupyter Notebook** – Coding and analysis environment
* **PostgreSQL** – Database storage and SQL analysis
* **Power BI** – Interactive dashboard and visualization
* **Gamma** – Presentation creation
* **SQL** – Querying and analyzing structured data

---

## Project Steps

### 1. Data Loading

* Imported the dataset using **Python (Pandas)** in Jupyter Notebook.
* Loaded the data into a DataFrame for analysis.

### 2. Exploratory Data Analysis (EDA)

* Examined dataset structure using `.info()` and `.describe()`
* Identified trends and distributions
* Checked for missing or inconsistent values

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types where necessary
* Prepared the dataset for further analysis

### 4. SQL Analysis (PostgreSQL)

* Uploaded cleaned data to **PostgreSQL database**
* Performed analytical queries such as:

  * Total purchase amount
  * Category-wise sales analysis
  * Customer demographic insights

### 5. Data Visualization (Power BI)

* Connected Power BI to the dataset
* Built an **interactive dashboard** showing:

  * Purchase trends
  * Category distribution
  * Customer demographics
  * Sales by location

### 6. Reporting

* Documented the analysis findings in a **report**
* Created a **professional presentation using Gamma** summarizing insights.

## How to Run the Project

1. Clone or download the project repository.

2. Install required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

3. Open the **Jupyter Notebook** and run the data analysis script.

4. Import the cleaned dataset into **PostgreSQL** and execute the SQL queries.

5. Open the **Power BI file (.pbix)** to explore the interactive dashboard.

6. Review the **report and presentation** for a summary of findings.
